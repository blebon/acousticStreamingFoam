Welcome to acousticStreamingFoam's documentation!
=================================================

acousticStreamingFoam is an OpenFOAM implementation of an acoustic streaming 
solver based on `a nonlinear theory of sound propagation in bubbly liquids <https://doi.org/10.1016/j.ultsonch.2018.04.014>`_.

You will find the application source code on 
`Github <https://github.com/blebon/acousticStreamingFoam/blob/master/applications/solvers/helmholtz/acousticStreamingFoam>`_. 

A tutorial case corresponding to the geometry described in T. Subruto, G.S.B. Lebon, D.G. Eskin, I. Skalicky, D. Roberts, I. Tzanakis
and K. Pericleous, "Numerical modelling and experimental validation of the effect of ultrasonic melt treatment in a direct-chill cast 
AA6008 alloy billet", *Journal of Materials Research and Technology* **12** (2021) 1582-1596 `doi:10.1016/j.jmrt.2021.03.061 <https://doi.org/10.1016/j.jmrt.2021.03.061>`_ 
is found `here <https://github.com/blebon/acousticStreamingFoam/blob/master/tutorials/helmholtz/acousticStreamingFoam/USDC>`_ 
and documented on :doc:`tutorials/USDC`.
